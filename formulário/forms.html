Tudo que você precisa saber sobre formulários HTML!

O que é?

Um formulário HTML utiliza de elementos (widgets) para preencher dados ou resultar em uma ação desejada utilizando a linguagem de marcação HTML, ou seja, é uma forma de trazer a interação do usuário com a página.

Como criar?

A tag para criar um formulário é < form > , para a abertura, e < /form > , para o fechamento.

Dentro dela, é possível inserir os atributos "action", para definir o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário, e method, para definir o método HTTP com que o formulário HTML irá lidar com os dados recebidos.

Existem dois tipos de métodos: o método GET, onde os dados serão reconhecidos pelo servidor através da URL, e o método POST, onde os dados são enviados ao servidor através do corpo da solicitação HTTP.

É recomendado a utilização do método POST para o envio de dados confidenciais/pessoais e do GET para dados não seguros

Caso o atributo method não seja declarado, o formulário funcionará através do método GET, já que esse é o método padrão do formulário HTML.

Exemplo de uso da tag

< form method="insira aqui 'get' ou 'post'" action="insira aqui a URL que irá receber os dados" >

...

< /form >

Elementos dentro do Formulário

Para criar um formulário funcional, é necessário usufruir de elementos próprios do formulário, além de também ser possível utilizar outras tags "padrões" do HTML dentro da própria tag forms.

Dito isso, veja a seguir os elementos de formulário e suas funções:

< input > – Define um campo de entrada de dados;

< textarea > – Define uma área de texto, podendo conter diversas linhas de texto;

< button > – Define um botão;

< select > – Define uma lista selecionável, também conhecida como drop-down;

< option > – Define uma lista de opções dentro de um drop-down select;

< optgroup > – Define um grupo de opções;

< fieldset > – Define um grupo de campos;

< label > – Define um rótulo ou legenda para um campo ou controle do formulário;

< output > – Define elementos de saída para o formulário;

< legend > – Define um título para o conjunto de campos.
